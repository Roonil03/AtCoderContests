# A - Secret Numbers
### Time Limit: 2 sec / Memory Limit: 1024 MiB

## Score : 100 points

## Description:
You are given a string 
S consisting of lowercase English letters and digits.

Extract only the digit characters from 
S and output the string formed by arranging them in their original order.

## Constraints
- S is a string of length between 1 and 50, inclusive, consisting of lowercase English letters and digits.

## Input
The input is given from Standard Input in the following format:
```
S
```

## Output
Output the answer.

## Examples:
### Example 1:
**Sample Input:**
```
abc462
```
**Sample Output:**
```
462
```
### Example 2:
**Sample Input:**
```
codequeen
```
**Sample Output:**
```
```
**Explaination:**  
`codequeen` contains no digits, so output the empty string.

### Example 3:
**Sample Input:**
```
31415
```
**Sample Output:**
```
31415
```

### Example 4:
**Sample Input:**
```
10plus2is12
```
**Sample Output:**
```
10212
```
